### What is Binary Search Tree?

Binary Search Tree中文稱作二元搜索樹，</br>
一棵正確的binary search tree需要符合一些條件 :</br>
樹根的左節點只會存在比他小或相等的值；</br>
樹根的右節點只會存在比他大的值，</br>
也就是說Binary Search Tree 一定不會出現左節點大於右子樹的情況。</br>
且每個節點底下一定只會有小於或等於兩個子節點。</br>
